About

This application explores potential solutions through proof of concept demos allowing administrators to send targeted communications to students through an application on their mobile device. This would allow various administrative stakeholders (instructors, chairs, Deans, etc) to engage with students when it matters most–their phone.

The work represented here is the result of a design ethnography course at Boise State University (ETHNO 597: Fundamentals of Design Ethnography). The methods used to design these high-fidelity mock-ups were customer journey mapping, empathy maps, and stakeholder interviews.

Customer Journey Map:

Stages Pick a Channel Select a Target Audience Create Communication Analyzation
Goals Select a method for communicating with campus population Choose recipients of the communication based on campus demographics. Input the information to be sent to target audience. View analytics to gage effectiveness of the communication.
Touchpoints
  • Select “Activity Feed” channel
  • Select “Push Notification” channel
  • Filter by “All Students”
  • Filter by College
  • Input a headline
  • Input description
  • Input media type (audio, video, link, calendar event)
  • Input a link url
  • Input an image url
  • Show the total count of created items.
  • View list of all content items the logged in user created.
  • View detail of content analytics.
